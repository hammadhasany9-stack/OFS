# Medzah Order Fulfillment System

A Next.js 16 prototype for managing LabCorp kit orders.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 16 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons
